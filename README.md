# 📚 NoteNest — Student Resource Sharing Platform

> Share Notes. Learn Together. Achieve More.

NoteNest is a free mobile-first web platform designed to help students easily share and access academic study materials in one centralized place. Students can upload, browse, download and preview educational resources such as lecture notes, PDFs, assignments, and study guides — completely free.

---

## 🌐 Live Demo
https://notenest.vercel.app

---

## ✨ Features
- 🔐 User registration and login with email
- ⬆️ Upload study notes as PDF files
- 🔍 Search and filter notes by subject or keyword
- ⬇️ Download resources uploaded by other students
- 👁️ Preview PDF notes directly in the browser
- 📊 Personal dashboard to manage uploaded notes
- 📱 Fully responsive mobile-first interface
- ℹ️ About, Privacy Policy and Contact pages
- 🖱️ Smooth scroll and hover animations
